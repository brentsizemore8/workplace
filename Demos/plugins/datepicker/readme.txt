1、日期new Date()
	new Date(2017,10,16) =>  2017.11.16
	new Date(2017,0,16) =>  2017.1.16
	new Date(2017,12,16) =>  2018.1.16

	注意：月份-1

2、日期函数
	getFullYear()		返回当前年份
	getYear()			当前年距离1900年
	getMonth()			返回当前月份
	getDate()			返回当前几号
	getDay()			返回当前日期处于星期几


3、当月第一天			new Date(year,month-1,1)
   当月最后一天			new Date(year,month,0)
